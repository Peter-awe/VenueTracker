# AcademicTracker — Submission Management Dashboard

A single-file, offline tool for managing academic paper submissions, tracking deadlines, and discovering publication opportunities.

## Features
- **Paper-Venue Matching**: See which venues fit your papers
- **Special Issue Tracker**: Track CFPs with deadline countdowns
- **Deadline Calendar**: Visual timeline of all upcoming deadlines
- **Submission Status**: Track papers through the review pipeline
- **Venue Database**: Searchable database of journals and conferences
- All data persisted locally — no server needed

## Usage
1. Download `index.html`
2. Open in any browser
3. Edit the data arrays in the HTML to add your venues and papers

## 5 Views

| Tab | Purpose |
|-----|---------|
| Paper-Venue Match | Find the best venue for each paper |
| Special Issues | Track calls for papers with deadlines |
| Calendar | See all deadlines on a timeline |
| Submissions | Track paper status through review |
| Venues | Browse and search venue database |

## Customization
Edit `VENUES`, `MY_PAPERS`, `SUBMISSIONS`, and `DEADLINES` arrays in the HTML file.
